# Intelligent Investor Metrics (Graham)

## Core Principles
1. **Margin of Safety** - Buy at discount to intrinsic value
2. **Defensive Investor** - Focus on large, stable companies
3. **Earnings Stability** - Consistent earnings over 10+ years
4. **Dividend Record** - Consistent dividends show financial health

## Key Metrics

### 1. Financial Strength (Defensive)
- Current Ratio > 2.0 (Graham: > 2.0)
- Debt/Equity < 0.5 (Graham: < 0.5)
- Quick Ratio > 1.0

### 2. Earnings (Value)
- P/E < 15 (Graham: < 15)
- P/E vs S&P 500 < 0.8 (should be lower than market)
- E/P (Earnings Yield) > 6.67% (inverse of 15 P/E)
- EBIT/EV > 6% (Moore variant)

### 3. Growth & Stability
- EPS Growth (10y) > 0%
- EPS Stability (coefficient of variation) < 0.25
- Revenue Growth (5y) > 0%

### 4. Dividend
- Dividend Yield > 2.5% (Graham preferred)
- Dividend Payout < 60%
- Dividend Growth (5y) > 0%

### 5. Valuation (Graham Number)
- Graham Number = sqrt(22.5 x EPS x Book Value per Share)
- Price/Graham Number < 0.5 = Undervalued
- Price/Graham Number > 1.5 = Overvalued

### 6. Deep Value (NCAV)
- NCAV = Current Assets - Total Liabilities
- P/NCAV < 1.0 = Significant margin of safety
- P/NCAV < 0.5 = Very cheap
